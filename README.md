## 先聊聊它到底是什么

如果你听说过 Notion、Obsidian、Roam Research，可以把思源理解成"把这几家的精华捏在一起，然后数据放本地"的版本。

核心逻辑是**块编辑**——每段话、每个列表项都是独立的"块"，像乐高一样可以随意拖动、引用、嵌套。在这个基础上，思源做了**双向链接**：A 文档里引用了 B 文档的某一段，B 那边会自动出现反向链接，告诉你"谁引用了我"。时间长了，你的知识库就会自然长出一张网来，而不是一堆孤立的文件夹。

支持平台：Windows、macOS、iOS、Android，全覆盖。本地离线随时可用，买了订阅的还能云端同步。

**双向链接 + 反向链接**：写一篇"机器学习笔记"，里面引用了另一篇"线性代数笔记"的某一段——后者会自动记录这条引用。过几个月回头看，你会发现自己无意间建了一张知识地图。

**端到端加密同步**：加密在本地完成，密钥由你自己掌握。云端存的是密文，开发者自己也解不了。对于记录工作内容、个人私密想法的用户来说，这一点非常关键。
